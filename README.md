# buggy-translate
a browser extension to translate words selected in  page

这是一个翻译单词的浏览器扩展．看网页时用鼠标选中一个单词，然后插件会在一个浮动框中显示翻译结果．

[扩展安装页面][1] https://addons.mozilla.org/zh-CN/firefox/addon/buggy-translate/    


## build
web-ext build
